## 🔬 جزئیات فنی

### **تکنولوژی استفاده شده:**
- 🧠 **Tesseract OCR 4.1+** با پشتیبانی فارسی
- 🎨 **PIL/Pillow** برای پیش‌پردازش تصویر
- ⚡ **Concurrent Processing** برای سرعت بالا
- 📄 **pdf2image** برای تبدیل PDF

### **الگوریتم‌های پیش‌پردازش:**
- تبدیل به Grayscale
- افزایش کنتراست (2x)
- بهینه‌سازی وضوح
- حذف نویز

### **بهینه‌سازی عملکرد:**
- پردازش موازی با 3 worker
- مدیریت حافظه هوشمند
- پردازش دسته‌ای صفحات
- پاکسازی خودکار فایل‌های موقت
